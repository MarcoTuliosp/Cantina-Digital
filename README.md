# 🍱 Cantina Digital

> Sistema de pedidos para cantina — escalável para sorveteria e açougue.

## Stack

| Camada | Tecnologia | Hospedagem |
|--------|-----------|------------|
| **Frontend** | React + Vite + TypeScript | Vercel (Free) |
| **Backend** | Java 17 + Spring Boot 3 | Render (Free) |
| **Banco** | PostgreSQL | Neon (Free) |
| **Automação** | n8n | Render (Futuro) |

## Estrutura do Projeto

```
cantina-digital/
├── 📋 docs/         → Documentação do projeto
├── 🔧 backend/      → API Java Spring Boot
├── 🌐 frontend/     → Interface React + Vite
├── ⚙️ infra/         → Configs de infraestrutura cloud
└── README.md
```
